あなたと共に
成長するエージェント。

IDEに紐づくコーディングアシスタントでも、単一APIのチャットボットラッパーでもありません。 あなたのサーバーに常駐し、学んだことを記憶し、稼働するほどに賢くなる 自律型エージェントです。

$ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

Linux、macOS、WSL2で動作 · 前提条件なし · すべて自動インストール

Nous Portalに登録

60秒でスタート

1

インストール

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

uv、Python 3.11をインストールし、リポジトリをクローンしてすべてをセットアップします。sudoは不要です。

2

設定

bash 
# 対話式セットアップウィザード
hermes setup

# またはモデルを選択
hermes model

Nous Portal（OAuth）、OpenRouter（APIキー）、または独自のエンドポイントに接続します。

3

チャット開始

bash 
hermes

これだけです。ツール、メモリ、スキルを備えた完全な対話型CLIが起動します。

4

マルチプラットフォーム対応 （オプション）

bash 
# 対話式ゲートウェイセットアップウィザード
hermes gateway setup

# メッセージングゲートウェイを起動
hermes gateway

# システムサービスとしてインストール
hermes gateway install

Telegram、Discord、Slack、WhatsAppへの接続をガイドします。systemdサービスとして動作します。

5

最新状態を維持

bash 
hermes update

最新の変更をプルし、依存関係を再インストールします。いつでも実行して新機能やバグ修正を取得できます。

ネイティブWindows対応は非常に実験的で、サポート対象外です。 WSL2 をインストールし、そこからHermes Agentを実行してください。

動作デモ

hermes

機能

あなたのいる場所に

Telegram、Discord、Slack、WhatsApp、CLIを単一のゲートウェイから — ひとつで始めて、別のプラットフォームで続けられます。

使うほどに賢くなる

永続的なメモリと自動生成スキル — プロジェクトを学習し、問題の解決方法を決して忘れません。

スケジュール自動化

自然言語によるcronスケジューリングでレポート、バックアップ、ブリーフィングを自動実行 — ゲートウェイを通じて無人で動作します。

委譲と並列実行

独自の会話、ターミナル、Python RPCスクリプトを持つ分離されたサブエージェントで、コンテキストコストゼロのパイプラインを実現します。

本格的なサンドボックス

ローカル、Docker、SSH、Singularity、Modalの5つのバックエンド — コンテナ強化とネームスペース分離を備えています。

Web・ブラウザ制御

Web検索、ブラウザ自動操作、ビジョン、画像生成、テキスト読み上げ、マルチモデル推論に対応。

ツール

40以上の組み込みツール — Web検索、ターミナル、ファイルシステム、ブラウザ自動操作、ビジョン、画像生成、テキスト読み上げ、コード実行、サブエージェント委譲、メモリ、タスク計画、cronスケジューリング、マルチモデル推論など。

プラットフォーム

Telegram、Discord、Slack、WhatsApp、Signal、Email、CLI — すべて単一のゲートウェイから。 Nous Portal、OpenRouter、または任意のOpenAI互換APIに接続可能。

実行環境

ローカル、Docker、SSH経由、Modal、Daytona、またはSingularityで実行。読み取り専用ルート、ケイパビリティ削除、ネームスペース分離によるコンテナ強化。

スキル

MLOps、GitHubワークフロー、リサーチなどをカバーする40以上のバンドルスキル。エージェントはオープンな agentskills.io フォーマットで新しいスキルを動的に作成・共有。 ClawHub、 LobeHub、GitHubからコミュニティスキルをインストール可能。

研究

並列ワーカーとチェックポイントによるバッチトラジェクトリ生成。RL訓練のためのAtropos統合。トラジェクトリ圧縮によるShareGPTフォーマットでのファインチューニング用エクスポート。
